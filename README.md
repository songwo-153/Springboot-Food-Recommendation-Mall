# 基于springboot美食推荐商城的设计与实现

#### 介绍
基于springboot美食推荐商城的设计与实现，后端框架使用Springboot+mybatis，前端框架使用Vue+hrml，数据库使用mysql，系统分为用户端前台系统和管理员端管理系统。

源码获取：[Springboot美食推荐商城系统源码]()

## 基于springboot的网上服装商城推荐系统的设计与实现
**开发语言：Java
框架：springboot
JDK版本：JDK1.8
服务器：tomcat7
数据库：mysql 5.7
数据库工具：Navicat11
开发软件：idea**

**项目介绍:**
**后端框架使用Springboot+mybatis
前端框架使用Vue+hrml
数据库使用mysql**
